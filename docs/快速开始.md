# 3 分钟快速开始

## 第一步：安装 Python

如果你电脑上已经有 Python，跳到第二步。

1. 打开 https://www.python.org/downloads/
2. 点击 "Download Python 3.x.x"
3. 运行安装程序
4. **重要：勾选 "Add Python to PATH"**
5. 点击 "Install Now"

## 第二步：安装依赖

打开命令行：

- **Windows**: 按 `Win + R`，输入 `cmd`，回车
- **Mac**: 打开 "终端" 应用

输入以下命令：

```bash
cd D:\python-toolkit
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

等待安装完成（约1-2分钟）。

## 第三步：开始使用

### 示例1：合并 PDF

```bash
python tools/pdf_processor.py merge --input 文件1.pdf 文件2.pdf --output 合并.pdf
```

### 示例2：抓取网页内容

```bash
python tools/web_scraper.py --url https://news.qq.com --mode article --output 新闻
```

### 示例3：整理下载文件夹

```bash
python tools/file_organizer.py by-type --dir C:\Users\你的用户名\Downloads
```

### 示例4：批量重命名照片

```bash
python tools/file_rename.py rename --dir D:\照片 --pattern "{date}_{num}" --start 1
```

---

## 遇到问题？

1. 确保 Python 已安装且添加到 PATH
2. 确保在工具包目录下运行命令
3. 查看 `docs/使用说明.md` 获取详细帮助
