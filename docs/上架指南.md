# 上架指南 — 怎么把这个工具包卖出去

## 方案一：Gumroad（海外，推荐）

### 步骤
1. 注册 Gumroad 账号：https://gumroad.com
2. 点击 "Create a product"
3. 上传 `python-toolkit` 整个文件夹（压缩成 zip）
4. 填写产品信息（用销售文案.md 里的英文版）
5. 设置价格：$9.99
6. 发布

### 优势
- 自动处理付款
- 全球用户
- 平台帮你做营销

---

## 方案二：小报童（国内，推荐）

### 步骤
1. 注册小报童账号：https://www.xiaobot.net
2. 创建新专栏
3. 上传工具包（可以分章节介绍每个工具）
4. 设置价格：¥29.9
5. 发布

### 优势
- 国内用户付费习惯好
- 微信支付，收款方便
- 平台抽成低

---

## 方案三：GitHub + 赞助

### 步骤
1. 创建 GitHub 仓库，上传工具包
2. 写好 README（用 README.md）
3. 开启 GitHub Sponsors
4. 在 README 里加上赞助链接

### 优势
- 建立个人品牌
- 长期收益
- 开源可以吸引更多用户

---

## 方案四：闲鱼/淘宝

### 步骤
1. 把工具包打包成 zip
2. 闲鱼发布商品
3. 标题："Python自动化工具包 12个脚本 文件处理/爬虫/Excel"
4. 价格：¥19.9 - ¥29.9
5. 买家付款后发送 zip 文件

### 优势
- 国内最大二手平台
- 用户量大
- 无需技术知识

---

**短期赚钱（1-3天见效）：**
- 闲鱼发布（最快）
