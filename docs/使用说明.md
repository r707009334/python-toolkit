# Python 自动化工具包 — 使用说明

## 安装

### 1. 安装 Python

如果你还没有安装 Python，请从 [python.org](https://www.python.org/) 下载并安装 Python 3.8 或更高版本。

安装时请勾选 **"Add Python to PATH"**。

### 2. 安装依赖

打开命令行（Windows 按 `Win+R` 输入 `cmd`），进入工具包目录：

```bash
cd D:\python-toolkit
pip install -r requirements.txt
```

### 3. 验证安装

```bash
python tools/pdf_processor.py --help
```

如果显示帮助信息，说明安装成功。

## 工具使用详解

---

### 1. PDF 处理器 (`pdf_processor.py`)

#### 合并多个 PDF

```bash
python tools/pdf_processor.py merge --input 文件1.pdf 文件2.pdf 文件3.pdf --output 合并结果.pdf
```

#### 拆分 PDF (每页一个文件)

```bash
python tools/pdf_processor.py split --input 大文件.pdf --output-dir ./拆分结果 --pages 1
```

#### 拆分 PDF (每5页一个文件)

```bash
python tools/pdf_processor.py split --input 大文件.pdf --output-dir ./拆分结果 --pages 5
```

#### 提取指定页面

```bash
# 提取第1、3、5到8页
python tools/pdf_processor.py extract --input 原文件.pdf --pages "1,3,5-8" --output 提取结果.pdf
```

#### PDF 转图片

```bash
python tools/pdf_processor.py toimage --input 文件.pdf --output-dir ./图片 --format png --dpi 200
```

---

### 2. 网页数据抓取 (`web_scraper.py`)

#### 抓取网页文章

```bash
python tools/web_scraper.py --url https://example.com/article --mode article --output 文章
```

#### 抓取网页表格

```bash
python tools/web_scraper.py --url https://example.com/data --mode table --output 数据 --format csv
```

#### 抓取网页列表

```bash
# 使用 CSS 选择器
python tools/web_scraper.py --url https://example.com --mode list --selector "ul li a" --output 链接
```

#### 抓取网页所有链接

```bash
python tools/web_scraper.py --url https://example.com --mode links --output 链接 --format csv
```

#### 处理中文乱码

如果网页有乱码，指定编码：

```bash
python tools/web_scraper.py --url https://example.com --mode article --encoding gbk --output 文章
```

### 3. 批量文件重命名 (`file_rename.py`)

#### 按序号重命名

```bash
python tools/file_rename.py rename --dir ./photos --pattern "照片_{num}" --start 1
```

结果: `照片_0001.jpg`, `照片_0002.jpg`, ...

#### 按日期重命名

```bash
python tools/file_rename.py rename --dir ./photos --pattern "{date}_{name}"
```

结果: `2026-06-27_IMG_001.jpg`, ...

#### 支持的变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `{name}` | 原文件名 | `photo` |
| `{ext}` | 扩展名 | `jpg` |
| `{date}` | 修改日期 | `2026-06-27` |
| `{num}` | 序号 | `0001` |
| `{size}` | 文件大小(KB) | `256` |

#### 添加前缀

```bash
python tools/file_rename.py affix --dir ./docs --prefix "2026_"
```

#### 预览模式 (不实际修改)

```bash
python tools/file_rename.py rename --dir ./photos --pattern "{num}_{name}" --dry-run
```

### 4. 文件夹整理 (`file_organizer.py`)

#### 按文件类型分类

```bash
python tools/file_organizer.py by-type --dir ~/Downloads
```

自动创建文件夹: `图片/`, `文档/`, `视频/`, `音频/`, `压缩包/`, `代码/`, `其他/`

#### 按日期分类

```bash
python tools/file_organizer.py by-date --dir ~/Downloads --format "%Y-%m"
```

自动创建: `2026-06/`, `2026-05/`, ...

#### 清理空文件夹

```bash
python tools/file_organizer.py clean --dir ~/Downloads
```

#### 预览模式

```bash
python tools/file_organizer.py by-type --dir ~/Downloads --dry-run
```

### 5. Excel 处理器 (`excel_processor.py`)

#### 合并多个 Excel

```bash
python tools/excel_processor.py merge --input 销售1月.xlsx 销售2月.xlsx 销售3月.xlsx --output 一季度.xlsx
```

#### Excel 转 CSV

```bash
python tools/excel_processor.py to-csv --input 数据.xlsx --output 数据.csv
```

#### CSV 转 Excel

```bash
python tools/excel_processor.py to-excel --input 数据.csv --output 数据.xlsx
```

#### 筛选数据

```bash
# 筛选某列大于100的行
python tools/excel_processor.py filter --input 数据.xlsx --output 筛选结果.xlsx --column "销售额" --condition gt --value 100

# 筛选包含某文本的行
python tools/excel_processor.py filter --input 数据.xlsx --output 筛选结果.xlsx --column "姓名" --condition contains --value "张"
```

#### 数据分析

```bash
python tools/excel_processor.py analyze --input 数据.xlsx --column "销售额"
```

#### 去除重复行

```bash
python tools/excel_processor.py dedup --input 数据.xlsx --output 去重结果.xlsx
```

### 6. 图片批处理 (`image_processor.py`)

#### 批量调整尺寸

python tools/image_processor.py resize --dir ./photos --output ./resized --width 800

#### 批量压缩

```bash

```

#### 批量格式转换

```bash
python tools/image_processor.py convert --dir ./photos --output ./png_images --format png
```

#### 批量添加水印

```bash
python tools/image_processor.py watermark --dir ./photos --output ./watermarked --text "版权所有" --position bottom-right
```

### 7. 文本批量替换 (`text_replacer.py`)

#### 批量替换文本

```bash
python tools/text_replacer.py replace --dir ./docs --old "旧公司名" --new "新公司名" --ext ".txt,.md"
```

#### 正则表达式替换

```bash
# 替换所有电话号码格式
python tools/text_replacer.py regex --dir ./docs --pattern "(\d{3})-(\d{4})-(\d{4})" --replacement "(\1)\2-\3"
```

#### 搜索关键词

```bash
python tools/text_replacer.py find --dir ./docs --keyword "重要" --ext ".txt,.md"
```

#### 批量下载文件

```bash

```

## 常见问题

### Q: 提示 "python 不是内部或外部命令"

说明 Python 没有添加到 PATH。解决方法：
1. 重新安装 Python，勾选 "Add Python to PATH"
2. 或手动添加 Python 到系统环境变量

### Q: 提示 "pip 不是内部或外部命令"

尝试：
```bash
python -m pip install -r requirements.txt
```

### Q: 安装依赖很慢

使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: PDF 转图片失败

需要安装 poppler：
- Windows: https://github.com/osber/poppler-windows/releases
- macOS: `brew install poppler`
- Linux: `sudo apt install poppler-utils`

### Q: 网页抓取有乱码

指定编码：
```bash
python tools/web_scraper.py --url ... --encoding gbk
```

## 自定义修改

所有脚本都是纯 Python 编写，你可以：

1. 修改文件路径和默认参数
2. 添加新的处理逻辑
3. 集成到自己的项目
4. 作为模块导入使用

```python
# 示例：在自己的代码中使用
from tools.pdf_processor import merge_pdfs
from tools.excel_processor import merge_excels

merge_pdfs(["file1.pdf", "file2.pdf"], "merged.pdf")
```
