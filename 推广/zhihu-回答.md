## 回答模板

作为一个经常和文件打交道的人，我写了一套 Python 自动化工具包，包含 12 个即用脚本，覆盖日常 90% 的文件处理需求。

**直接看效果：**

```bash
# 合并 3 个 PDF，一条命令搞定
python tools/pdf_processor.py merge --input 合同1.pdf 合同2.pdf 合同3.pdf --output 合并合同.pdf

# 下载文件夹一键整理（按类型分到不同文件夹）
python tools/file_organizer.py by-type --dir ~/Downloads

# 批量重命名 500 张照片
python tools/file_rename.py rename --dir ./photos --pattern "{date}_{num}" --start 1
```

**包含的 12 个工具：**

1. PDF 合并/拆分/提取/加水印/转图片
2. 网页数据抓取（表格、文章、链接）
3. 批量文件重命名
4. 文件夹自动整理（按类型/日期）
5. Excel 合并/筛选/转换/分析
6. 图片批量压缩/格式转换/加水印
7. 文本批量替换（支持正则）
8. 批量下载器
9. 文本分析/关键词提取

**为什么做这个？**

之前工作中经常遇到这些场景：几十个 PDF 要合并、下载文件夹乱成一团、Excel 表格要汇总……每次都手动操作太浪费时间。索性写了一套工具，现在一条命令就能搞定。

**怎么用？**

零基础也能用。解压后安装依赖（pip install -r requirements.txt），然后复制命令改改参数就行。每个工具都有详细中文说明。
