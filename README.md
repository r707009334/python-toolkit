## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行单个工具

```bash
# PDF 合并
python tools/pdf_processor.py merge --input 文件1.pdf 文件2.pdf --output 合并.pdf

# 网页抓取
python tools/web_scraper.py --url https://example.com --output data.csv

# 文件批量重命名
python tools/file_rename.py --dir ./photos --pattern "日期_序号"
```

### 3. 查看帮助

```bash
python tools/pdf_processor.py --help
```
